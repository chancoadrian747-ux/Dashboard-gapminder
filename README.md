Dashboard Gapminder
Los datos son de Gapminder y cubren el periodo 1972 a 2007.
Qué incluye :
- Mapa mundial con burbujas (color = esperanza de vida, tamaño = población)
- Gráfico de tendencias por continente
- Proyección con regresión lineal (continente americano hasta 2025)
- Relación entre PIB y esperanza de vida
- Tabla de datos descargable
- Filtros por año y continente
